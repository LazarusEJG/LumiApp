#Description

This is the LumiApp

This is designed as a full standalone application that runs llama-server from llama.cpp.
It also makes use of the dlls from llama.cpp

#instructions

get llama.cpp base dlls (ggml-base.dll, ggml-cpu-x64.dll (or cuda / other llama.cpp), ggml.dll ) together with llama-server.exe
get a model with the .gguf extension

place all of these files in the backend folder
resources/backend

run the application after ensuring the location of the files is visible to the frontend.
